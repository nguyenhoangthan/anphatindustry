1 Tổng Quan Dự Án

Website anphatindustry.com là trang web doanh nghiệp (corporate website) cho Công ty An Phát Industry – trung tâm bảo dưỡng, sửa chữa và chăm sóc ô tô chuyên nghiệp tại TP. Hồ Chí Minh. Website được xây dựng với mục tiêu:

🎯 Mục Tiêu Chính

Giới thiệu thương hiệu, dịch vụ và tạo kênh tiếp nhận khách hàng trực tuyến 24/7

📈 Mục Tiêu SEO

Xuất hiện top Google cho từ khóa "bảo dưỡng ô tô HCM", "sửa chữa ô tô uy tín"

📱 Thiết Bị Hỗ Trợ

Responsive 100%: Mobile, Tablet, Desktop. Tối ưu cho Chrome, Safari, Firefox, Edge

🚀 Hiệu Năng

Tốc độ tải trang dưới 3 giây. Tất cả 25 trang được render tĩnh (SSG) – cực kỳ nhanh

✅ Tham Khảo Thiết Kế Website được xây dựng theo cấu trúc và phong cách của thanhphongauto.com – một trong những website garage ô tô uy tín hàng đầu tại TP.HCM. Màu sắc chủ đạo: Xanh Navy #0B2447 (chuyên nghiệp) và Đỏ #E63312 (năng động, nổi bật CTA).

8.3 Deploy Lên Hosting Matbao

  • 1

    Đăng ký VPS Matbao

    Vào matbao.net, mua gói VPS Ubuntu 22.04. Nhận IP, username, password qua email.

  • 2

    Đăng ký tên miền

    Mua anphatindustry.com + anphatindustry.vn tại matbao.net để bảo vệ thương hiệu.

  • 3

    Cài môi trường server

    SSH vào VPS → cài Node.js 20 → cài PM2 (process manager) → cài Nginx (reverse proxy).

  • 4

    Upload source code

    Clone repo hoặc upload toàn bộ thư mục lên server → chạy npm install → npm run build.

  • 5

    Khởi động ứng dụng với PM2

    pm2 start npm --name "anphatindustry" -- start → pm2 save

  • 6

    Cài SSL miễn phí (Let's Encrypt)

    Cài Certbot → certbot --nginx -d anphatindustry.com -d www.anphatindustry.com

  • 7

    Trỏ DNS về server

    Vào matbao.net quản lý DNS → thêm A record trỏ domain về IP VPS → chờ 1–24h lan truyền.

8.4 Deploy Nhanh Miễn Phí (Vercel – Khuyến Nghị Thử Trước)

💡 Cách nhanh nhất để xem website live Đăng ký tài khoản tại vercel.com → Import repo → Deploy tự động. Nhận URL dạng anphatindustry.vercel.app miễn phí để test trước khi kết nối domain chính thức.
