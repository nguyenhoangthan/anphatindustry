## Giới thiệu

Kính gửi Quý Công ty, đây là tài liệu bàn giao **website chính thức của Công Ty Cổ Phần An Phát** — trung tâm bảo dưỡng & sửa chữa ô tô, cung cấp mô hình thiết bị đào tạo kỹ thuật và giải pháp chẩn đoán.

Website được xây dựng trên nền tảng công nghệ hiện đại (Next.js), giao diện chuẩn thương hiệu, tối ưu hiển thị trên cả máy tính và điện thoại. Đặc biệt, Quý Công ty **toàn quyền tự cập nhật nội dung** (dịch vụ, bài viết, hình ảnh, video, thông tin liên hệ…) thông qua **trang quản trị** — đơn giản, không cần lập trình viên.

Tài liệu này hướng dẫn cách đăng nhập, những gì có thể chỉnh sửa, và các thông tin kỹ thuật để vận hành lâu dài.

## 3. Trang quản trị chỉnh sửa được những gì?

Quý Công ty có thể tự chỉnh sửa **gần như toàn bộ nội dung** hiển thị trên website:

| Khu vực trên website | Có thể chỉnh sửa |
|---|---|
| **Trang chủ** | Ảnh + chữ slider đầu trang (Hero); đoạn giới thiệu & số liệu; khối “Tại sao chọn chúng tôi”; quy trình dịch vụ; đối tác & số liệu; khối cam kết/kêu gọi (CTA). |
| **Dịch Vụ** | Thêm / sửa / xoá từng dịch vụ: tên, mô tả ngắn & chi tiết, danh mục (Bảo dưỡng & sửa chữa / Đồng sơn / Chăm sóc & làm đẹp / Hỗ trợ / Xe điện), ảnh, điểm nổi bật. |
| **Mô Hình Đào Tạo** | Hạng mục (Thiết bị đào tạo / chẩn đoán), từng nhóm và danh sách hạng mục con, mô tả, ảnh. |
| **Dự Án & Đối Tác** | Danh sách dự án đã thực hiện (tên, năm, mô tả, ảnh) và danh sách đối tác / khách hàng. |
| **Thư Viện** | Bài viết theo chuyên mục (Chia sẻ kinh nghiệm / Bảng tin kỹ thuật / Tuyển dụng); **Video Clip** (link YouTube hoặc tải video). |
| **Trang Giới Thiệu** | Giới thiệu công ty, sứ mệnh – tầm nhìn – năng lực, đội ngũ nhân sự. |
| **Liên hệ & Cài đặt** | Tên công ty, hotline, email, mạng xã hội, mã số DN, và **các cơ sở (địa chỉ + bản đồ)** — đổi ở đây cập nhật toàn site. |
| **Hình ảnh & Video** | Tải ảnh trực tiếp từ máy ở mọi nơi có ảnh; thêm video bằng link YouTube hoặc tải file. |

*Phần cố định không sửa qua admin (cần lập trình viên): bố cục/giao diện tổng thể, logo, cấu trúc menu.*

## 4. Thao tác thường dùng

- **📷 Tải ảnh lên:** ở ô ảnh bất kỳ → bấm **“Tải ảnh lên”** → chọn ảnh từ máy (tự điền link), hoặc dán URL. Ảnh ≤ 5MB.
- **🎬 Thêm video:** vào **Video Clip (Thư Viện)** → **Thêm video** → **khuyên dùng** dán link YouTube; hoặc bấm **“Tải video”** cho clip ngắn (≤ 50MB). Hiện tại `/thu-vien/video`.
- **💾 Lưu:** bấm **Lưu** cuối form — nội dung cập nhật ngay trên website.

**Quy trình cập nhật CODE** (chỉ khi đổi giao diện/tính năng):
```bash
npm run build
git add -A && git add -f .next/ node_modules/.prisma/
git commit -m "mô tả" && git push origin main
npm run deploy        # CI tự FTP lên server
# → Plesk → Node.js → RESTART APP (bắt buộc)
```
Chi tiết: `docs/DEPLOYMENT.md`, `docs/OPERATIONS.md`.

## 6. Backup & Vận hành

- **Backup định kỳ:** tải `/httpdocs/prisma/prisma/dev.db` (toàn bộ nội dung) và thư mục `/httpdocs/public/uploads/` (ảnh/video đã tải lên).
- **Khởi động lại:** Plesk → Node.js → **Restart App**. Không gõ tay `npm start`.
- Sau khi deploy code, hoặc nếu site báo **503/maintenance** → chỉ cần **Restart App**.
